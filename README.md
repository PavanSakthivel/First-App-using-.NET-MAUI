#  First App using .NET MAUI

This is a basic cross-platform application built with **.NET MAUI (Multi-platform App UI)**.  
It runs on **macOS (Mac Catalyst), iOS, and Android**, and demonstrates fundamental MAUI UI components and event handling.
This is the basic app created when creating a .NET MAUI project using C#

---

##  Features

- ✔ Cross-platform UI built with **XAML**
- ✔ Simple **button counter** — tracks how many times the user clicks
- ✔ Runs on **Mac Catalyst, iOS Simulator, and Android Emulator**
---

##  Preview

The app displays:

- A header ("Hello, .NET MAUI!")
- A welcome message
- A button labeled **"Click me"**
- Each click increments a counter displayed on the button

---

##  Project Structure
MauiApp1/
├── Platforms/ # Platform-specific code (Android, iOS, macOS)
├── Resources/ # Images, fonts, raw assets
├── App.xaml # Application-wide UI settings
├── AppShell.xaml # App navigation container
├── MainPage.xaml # UI layout for the main page
├── MainPage.xaml.cs # Logic for button click counter
├── MauiProgram.cs # App startup configuration
└── MauiApp1.csproj # Project file
